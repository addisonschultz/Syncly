# Support

### Contact support

* **Email**: `support@syncly.com`
* **Live chat**: Monday to Friday, `09:00–18:00 UTC`
* **Status page**: `status.syncly.example`
* **Enterprise hotline**: `+1 (555) 010-2450`

### Response targets

* **P1 · Critical outage**: First response in `1 hour`
* **P2 · Major impact**: First response in `4 hours`
* **P3 · Minor issue**: First response in `1 business day`
* **P4 · Question or request**: First response in `2 business days`

### Before you contact support

Include these details in your request:

* Workspace name or ID
* A short issue summary
* Steps to reproduce
* Screenshots or error messages
* The affected date, time, and time zone

### Common support topics

* Calendar sync delays
* Missing reminders
* Shared calendar permissions
* Recurring event conflicts
* Import or export failures

### Self-serve options

* Start with the [Quickstart](../getting-started.md)
* Check [Community](community.md) for known workarounds
* Review the status page before reporting outages
